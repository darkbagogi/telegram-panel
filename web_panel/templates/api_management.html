API Dokümantasyonu

Tüm API isteklerinde Authorization header'ında API anahtarınızı gönderin:

Authorization: Bearer {{ current_user.api_key }}
POST /api/campaigns

Yeni bir kampanya oluşturur.

Request Body:
{
  "name": "Kampanya Adı",
  "source_group": "Kaynak Grup ID",
  "target_group": "Hedef Grup ID",
  "daily_limit": 50,
  "filter_type": "active"
}
Response:
{
  "success": true,
  "campaign_id": 123,
  "message": "Kampanya başarıyla oluşturuldu"
}
GET /api/campaigns

Tüm kampanyalarınızı listeler.

Response:
{
  "campaigns": [
    {
      "id": 123,
      "name": "Kampanya Adı",
      "status": "running",
      "progress": 45.5,
      "created_at": "2024-01-01T00:00:00Z"
    }
  ]
}
GET /api/stats

Response:

  "total_invites": 1250,
  "successful_invites": 980,
  "success_rate": 78.4,
  "active_campaigns": 3
}
